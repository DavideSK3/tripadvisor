Il progetto è stato sviluppato ed è funzionante sotto un sistema Ubuntu 14.04

Nel caso venga eseguito sotto un sistema Windows è necessario cambiare nel codice sorgente le parti in cui si gentiscono i file (in quanto su Window nei percorsi viene utilizzato '\' invece che '/')

In ogni caso prima dell'esecuzione è necessario modificare nel file web.xml il percorso in cui si può trovare il database. Se nulla è stato modificato nella cartella del progetto esso si troverà sotto dentro la cartella "TripAdvisor/db/"
